# dict.txt

The dictionary is a modified version of Steve Hanon's combined dicitonary from [This Website](https://stevehanov.ca/blog/index.php?id=8)

The characters used in the dictionary correspond to the following enums in phonemes.h:

The comma (,) and apostrophe (') are used to denote strong or weak emphasis.

The forward slash (/) is ignored.

## Parts of speech

After each word, part of speech is denoted with

| Letter | Part of Speech    |
|--------|-------------------| 
| N      | Noun              |
| h      | Hyphenated Noun   |
| p      | Plural Noun       |
| r      | Pronoun           |
| A      | Adjective         |
| V      | Verb              |
| v      | Adverb            |
| i      | Intransitive Verb |
| t      | Transitive Verb   |
| !      | Interjection      |
| P      | Preposition       |
| C      | Conjunction       |
| D      | Determiner        |
